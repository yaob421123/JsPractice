# 使用说明

刚学会jquery的时候 写的一个悬浮框插件demo


## html部分：
```bash
<div class='fd'></div
```


css部分：
...



## js部分：
导入`jquery`库和`Suspension.js`
```bash
$(function(){
	$(".fd").Suspension('middle');
});
```

## 参数说明

- middleright  默认右侧居中显示

- leftbottom 左下角显示

- rightbottom  右下角显示

- lefttop  左上角显示

- righttop  右上角显示

- middletop  顶部居中显示

- middlebottom  低部居中显示

- middleleft  左侧居中显示

- middleright  右侧居中显示

- middle  居中
